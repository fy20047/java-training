# JAVA-TRAINING

這是一個用來練習 Java 基礎語法與 OOP 觀念的專案，內容依照章節整理成不同的資料夾，方便學習與回顧。

---

## 📂 專案結構

JAVA-TRAINING/
├── chapter1/ # Java 基本語法（變數、流程控制、方法）
├── chapter2/ # 物件導向 OOP（類別、繼承、多型）
├── chapter3/ # 進階主題（封裝、抽象、介面）

---

## 📝 章節內容
- **Chapter 1**：基礎語法與數學函式  
  - 變數、型別轉換  
  - if/else, switch, 迴圈  
  - 基本數學運算  

- **Chapter 2**：物件導向 (OOP)  
  - 類別與物件  
  - 繼承、方法覆寫  
  - 多型 (Polymorphism)  

- **Chapter 3**：進階觀念  
  - 封裝 (Encapsulation)  
  - 抽象類別與方法  
  - 介面 (Interface)
